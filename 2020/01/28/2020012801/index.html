栈基本操作

TOS 栈顶(栈上最高地址的值)
TOS1 栈上第二高地址的值
TOS2 栈上第三高地址的值
…………

  • LOAD_NAME

    压栈

  • POP_TOP

  • ROT_TWO

  • ROT_THREE

  • ROT_FOUR

  • DUP_TOP

  • DUP_TOPX(count)

UNARY

栈顶元素弹出去，运算后压栈

  • UNARY_NEGATIVE

    取负

  • UNARY_NOT

    逻辑取反

  • UNARY_INVERT

    按位取反

BINARY

栈顶两元素弹出去，运算后压栈

  • BINARY_ADD

    加法

  • BINARY_SUBTRACT

    减法

  • BINARY_MULTIPLY

    乘法

  • BINARY_DIVIDE

    除法

  • BINARY_MODULO

    取模

  • BINARY_POWER

    幂运算

  • BINARY_LSHIFT

    左移

  • BINARY_RSHIFT

    右移

  • BINARY_AND

    与

  • BINARY_OR

    或

  • BINARY_XOR

    异或

  • BINARY_SUBSCR

    索引

DELETE

  • DELETE_NAME

跳转

  • COMPARE_OP(op)

    对栈顶的两个元素做op指定的比较操作，结果压栈

  • POP_JUMP_IF_TRUE(target)

    栈顶弹出，如果为true跳转到target

  • POP_JUMP_IF_FALSE(target)

    栈顶弹出，如果为false跳转到target
